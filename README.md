# Robo Editor

Pick a robot and "fix" articles by swapping in "better" words.

## How to play

This game is under active development.

Running this game requires Node JS.

1. Clone the repo
2. `npm install`
3. run 'node game.js'
4. Go to 'localhost:3000' to play
